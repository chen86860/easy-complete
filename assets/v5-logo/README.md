# v5 Logo

This folder contains a small logo set for a command line tool named `v5`.

- `v5-logo.svg`: square icon / avatar mark
- `v5-logo-horizontal.svg`: horizontal lockup for README headers and docs
- `v5-logo.png`: exported preview of the square logo
- `v5-logo-horizontal.png`: exported preview of the horizontal lockup

The SVGs are transparent around the mark and can be edited directly.
